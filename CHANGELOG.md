## [0.1.0] - 2026-02-25

### Added

- `Builder` component with full SDK lifecycle management
- `useBuilder` composable for programmatic builder control and multi-instance management
- Collaborative editing support via `shared` and `sessionId` props
- Vue events (`bb-save`, `bb-error`, `bb-session-started`, etc.) for idiomatic event handling
- Optional config callbacks (dual mechanism) for React-familiar patterns
- SDK type re-exports for consumer convenience
- Demo application with co-editing, multi-language, and multi-builder support
